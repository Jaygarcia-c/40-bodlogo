# Бодлогын агуулга (Монгол хэлээр)

Bomberman тоглоомын дүрмээр, бөмбөгүүд 3 секундийн дараа дэлбэрч хөрш эсүүдийг устгадаг. Тоглоомын `n` секундийн дараах байдал ямар байхыг ол. Эхний байдал нь өгөгдсөн байна.
